종목별 1년 가격

Automation

자동 갱신 방식

01

매일 밤 실행

GitHub Actions가 매일 23:30 KST에 실행되어 최신 데이터를 만듭니다.

02

1년 자료 생성

각 종목의 최근 약 260거래일 가격과 과거 기준 점수 이력을 생성합니다.

03

정적 웹 배포

생성된 JSON과 CSV를 포함한 site/ 폴더를 GitHub Pages로 배포합니다.

04

명확한 최종 판정

최종 후보, 관찰 대상, 제외, 데이터 오류를 분리해서 표시합니다.

1MonthFinder는 투자 조언이 아닌 데이터 기반 후보 선별 도구입니다.
