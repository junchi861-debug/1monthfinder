지수 매매/위클리 옵션 분리 구현 정리
작성 기준: 2026-05-28


1. 결정된 방향

GitHub Actions로 하루 한 번 돌리는 구조는 맞지 않는다.

앞으로는 PC 또는 스마트폰 자원으로 서버를 띄우고, 1분/5분봉 데이터를 반복 갱신하는 방식이 맞다.

현재 웹앱은 두 흐름으로 나눈다.

- 종목 신호: 국내 주식 추천/보유 종목 차트
- 지수 매매: KOSPI200 지수/선물 전용 매매 신호

위클리 옵션 전략은 옵션 전용 기준을 받은 만큼 별도 탭으로 붙인다.


2. 이번에 먼저 구현한 것

- 웹앱에 지수 매매 탭 추가
- 웹앱에 위클리 옵션 탭 추가
- 지수 매매 공식선 미리보기 차트 추가
- 위클리 옵션 프리미엄 전략선 차트 추가
- 30/50 기하이평, 5분봉 전환선/기준선, 피보나치 61.8/80.9/100/103/105 라인 표시 구조 추가
- 핵심 자리 이탈 후 직전 저가를 새 61.8%로 맞추고 100% 자리를 다시 계산하는 피보나치 리셋 구조 추가
- 리셋된 61.8%~100% 사이 중심라인과 프리미엄 1.5~1.8 콜 2~3계약 재시작 후보 추가
- 원하는 자리 근처 밑꼬리 확인을 중심라인 콜 재시작 신뢰도 상승 조건으로 추가
- 70% 1차 진입, 30% 추가 진입, 105% 손절, 70% 청산, 잔량 운용, 5분봉 하락 압력, 쌍바닥 최소 트라이, 100% 앞선 트라이 상태 흐름 정리
- 옵션 당일 손실 300만원 중단 원칙, 당일 50~100만원 수익 후 보수 모드, 하락 압력 구간 최소 물량 트라이, 100% 앞선 자리 트라이, 만기일 12시 이후 1.5%/프리 0.6~1.0/40% 손절 규칙, 1회 손절 30~50만원 감수, 3계약 분할진입, 70% 안전마진 물량/30% 수익극대화 물량, 급반등 +20~30% 최소 1계약 청산, -20~-25% 손실권 자리이동, 자리이동 2회 제한, 약손실 후 대기, -30% 손실선 홀딩, +35%/+45% 부분청산, 잔량 운용 구조 정리
- 알고리즘 설정 JSON 분리


3. JSON 위치

지수 매매 원본 설정:

- config/algorithms/index_trading.json

웹앱 표시용 설정:

- site/data/index_trading.json

기존 국내 주식 알고리즘:

- config/algorithms/market_signals.json
- config/charts/dashboard.json
- site/data/chart_config.json

위클리 옵션 전략:

- config/algorithms/weekly_options.json
- site/data/weekly_options.json

위클리 옵션 차트선:

- config/algorithms/weekly_options.json 안의 visual_chart.lines


4. 다음 구현 순서

1. 지수 5분봉 데이터 입력 포맷 확정
- datetime
- open
- high
- low
- close
- volume

2. 기술지표 계산 모듈 추가
- GMA
- Ichimoku tenkan/kijun
- recent 3-day high/low
- fib_618, fib_809, fib_100, fib_103, fib_105
- reset_fib_618, reset_fib_100
- reset_mid_618_100
- lower_wick_5m, lower_wick_ratio_5m

3. 상태 머신 구현
- WAIT
- WATCH_ENTRY
- ENTRY_70
- WICK_WATCH
- ADD_30

- FIBONACCI_RESET
- RESET_MID_CALL_PREPARE
- RESET_MID_WICK_CONFIRM
- SELL_70

4. 실시간 또는 반복 계산 서버
- PC: scripts/run_pc_server.ps1

5. 알림

- 추후 카카오톡/텔레그램 등 외부 알림


5. 전문가 답변을 기다리는 부분

- 105% 손절 확정 기준
- 103% 밑꼬리 확인 기준
- 30% 추가 진입 횟수 제한
- 기준선 청산 기준이 터치인지 종가인지
- 위클리 옵션 행사가 선택 기준
- 옵션 프리미엄/미결제약정/거래량 필터
- 핵심 자리 이탈 기준
- 자리이동 2회 카운트 방식
- 약손실 기준
- 피보나치 리셋용 새 고가/직전 저가 확정 기준
- 리셋 중심라인에서 2계약/3계약 선택 기준과 1.5~1.8 프리미엄 주문가 분할 기준
- 원하는 자리 밑꼬리의 종가 회복 기준, 밑꼬리 비율, 근접 허용 오차


6. 주의

현재 지수 매매 탭의 차트는 실제 장중 데이터가 아니라 공식선 미리보기다.

실제 매수, 매도, 경고 신호는 5분봉 데이터 입력과 상태 머신 구현 후 표시해야 한다.
